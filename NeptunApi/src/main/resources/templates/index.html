Neptun REST API
Request method Url path Web service Authorization
GET classrooms A termek listája engedélyezett
GET classroom/{ID} Id által kiválasztott terem adatai engedélyezett
GET departments Tanszékek listája engedélyezett
GET department/{ID} Id által kiválasztott tanszék adatai engedélyezett
GET /equipments Felszerelések listája engedélyezett
GET /equipment/{ID} Id által kiválasztott felszerelés adatai engedélyezett
GET /students Diákok listája engedélyezett
GET /student/{ID} Id által kiválasztott diák adatai engedélyezett
GET /teachers Tanárok listája engedélyezett
GET /teachers/{ID} Id által kiválasztott tanár adatai engedélyezett
GET /teachers/department/{ID} Id által kiválasztott tanár és tanszék engedélyezett
GET /courses Kurzusok listája engedélyezett
GET /course/{ID} Kurzus ID szerint engedélyezett
GET /student/course/{neptun_code}/{days} Diák kurzusai az adott napon engedélyezett
GET /teacher/course/DFGHJK/hétfő/teacher/course/{neptun_code}/{days} Tanár kurzusai az adott napon engedélyezett

DELETE /equipment/delete/{ID} Kiválasztott Felszerelés törlése jelszóval védendő
POST /students/add Új Diák felvétele jelszóval védendő

DELETE /students/delete/{ID} Kiválasztott Diák törlése jelszóval védendő

DELETE /teachers/delete/{ID} Kiválasztott Tanár törlése jelszóval védendő

PATCH /course/updatename/update/{ID} Kurzus nevének módosítása jelszóval védendő
DELETE /course/delete/{ID} Kiválasztott Kurzus törlése jelszóval védendő
